# HonokaIndex
个人站点[honoka.cc](http://honoka.cc)
## 概述
目前含有
- [ttmail](http://honoka.cc/ttmail)
一个极简单的邮件追踪工具，原理类似于ga
- [ooec](http://hg.honoka.cc)
一个简单的cms,用于学校实验室内容展示

## 环境
- Ngnix 1.8
- PHP 7.1
- MySQL 5.6
- Redis 3.0
- Laravel 5.3

## 开发与部署
自动化部署，采用 github + capistrano3 + capistrano-laravel 方式

## 扩展包
- laravel-debugbar
  一个Laravel调试工具
